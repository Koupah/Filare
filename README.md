# Configs
A lightweight solution to creating configuration files for Spigot and BungeeCord plugins.

### Code Examples
```java
// Bukkit
BukkitConfigHelper bukkitConfig = new BukkitConfigHelper(this, "modules", this.getDataFolder().getAbsolutePath());
```
